# LINGUS - Plataforma de Contratação Inteligente de Professores Freelancer

## 📚 Sobre o Projeto

LINGUS é uma plataforma 100% digital e mobile-first que conecta escolas aos melhores educadores do mundo de forma inteligente, segura e acessível.

### Desafios Resolvidos

- ✅ **Contratação Lenta** → Processo rápido e eficiente
- ✅ **Falta de Qualidade** → Matching inteligente com IA
- ✅ **Custos Altos** → Sem intermediários, menor custo

### Principais Características

- 🤖 **Matching Inteligente com IA** - Recomendações baseadas em compatibilidade
- 📱 **Plataforma 100% Digital e Mobile-First** - Acesso de qualquer lugar
- 💳 **Sem Intermediários** - Menor custo, melhor preço
- ⭐ **Sistema de Reputação Transparente** - Reviews e ratings confiáveis
- 🔒 **Pagamento Seguro e Garantido** - Transações protegidas
- 🔗 **Integração com Sistemas Escolares** - Compatibilidade com plataformas educacionais

## 🎯 Fluxos Principais

### Fluxo da Escola
1. **Cria Pedido** - Define matéria, série, horas, valor
2. **Publica** - Abre para candidaturas
3. **Recebe Candidatos** - Visualiza professores qualificados com ratings

### Fluxo do Professor
1. **Completa Perfil** - Qualificações e disponibilidade
2. **Vê Oportunidades** - Filtros inteligentes
3. **Se Candidata** - Proposta de trabalho
4. **É Contratado** - Inicia trabalho e recebe pagamento direto

## 💼 Modelo de Negócio

| Modelo | Descrição |
|--------|-----------|
| **Comissão por Contratação** | 8-12% sobre cada contratação realizada |
| **Plano Premium Escolas** | Acesso ilimitado com visibilidade destacada |
| **Plano Premium Professores** | Visibilidade+ para ter mais oportunidades |

## 🛠️ Stack Tecnológico

### Frontend
- **React** - UI library
- **React Router** - Navigation
- **Axios** - HTTP client
- **TailwindCSS** - Styling
- **Redux/Context** - State management

### Backend
- **Node.js + Express** - Server
- **MongoDB** - Database
- **JWT** - Authentication
- **Stripe** - Payments
- **Socket.io** - Real-time communication

## 🚀 Como Começar

### Pré-requisitos
- Node.js 16+
- npm ou yarn
- MongoDB local ou cloud

### Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

### Setup Backend
```bash
cd backend
npm install
cp .env.example .env
# Configure suas variáveis de ambiente
npm run start
```
